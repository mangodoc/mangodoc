# mangodoc-cli脚手架
## 说明
`mangodoc-cli`是基于`mangodoc`的脚手架，基于`mangodoc`的模板仓库快速生成项目，并支持更新`cdn`和组件版本等功能。

## 安装
``` shell
npm install -g mangodoc-cli
```
查看版本
``` shell
mangodoc -v
0.1.1
```

## 使用
### 生成项目
``` shell
mangodoc init
``` 
### 更新cdn
``` shell
mangodoc update cdn
```
### 更新组件版本
``` shell
mangodoc update version
```
